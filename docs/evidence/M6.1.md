## Surprises / deviations from the plan

- **uvicorn is a grandchild of `uv run`**, so killing the recorded PID left the port
  bound. Fix: start each process with `start_new_session=True` (its own group) and
  `os.killpg` on `down`, then wait for the socket to actually release (SIGTERM is async).
- **The full workspace suite hangs on THIS box** — not a code failure: ~30 tests each
  spin up their own Anvil, and on a 14 GB machine already holding other users' Ollama
  models, six concurrent Anvils thrash. Every milestone's suite passes in isolation and
  CI (a fresh runner) runs the whole workspace green; the hang is the shared box, and it
  is why `just up` runs Anvil once and the lab stays opt-in.
