## Surprises / deviations from the plan

- **A container subscription yields ONE value, not per-leaf updates**: subscribing to
  `…/statistics` returns a single update whose value is the whole statistics dict. The
  first assertion looked for `in-octets` among the value *keys* and failed while the
  data was sitting one level down.
- `TelemetrySample` is a genuinely new cross-package shape (forwarder → collector), so
  it landed in `a2a_interfaces` + docs/03 §5.1 in this same commit (rule 3); it carries
  its own `"v": 0`.
- The plan's "dummy collector (netcat or 20-line asyncio server)" became
  `netctl.testing.DummyCollector` (25 lines, threads) — reused by the contract suite,
  the lab test, and the notebook.
