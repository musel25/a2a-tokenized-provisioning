## Surprises / deviations from the plan

- The plan said "schema added to interfaces, `v` bump". `DashboardEvent` is **additive**
  (a new shape, not a change to an existing one), so it carries its own `"v": 0` like
  `Decline`/`TelemetrySample` — no global bump, nothing breaks. docs/03 §8 documents it.
- The interactive UI (columns, stepper) is watched via `streamlit run`; the *test* covers
  the event path and the app's importability, since a browser can't live in CI. The
  headless boot (HTTP 200) confirms the app itself is sound.
