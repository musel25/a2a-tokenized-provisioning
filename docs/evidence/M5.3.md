  crosses the A2A wire, so it belongs there (rule 3), added with docs/03 in this commit.

  because it never crosses a boundary — only its result (`SignedOffer` or `Decline`) does.
- The two-kinds-of-no distinction sharpened the design: admission (physical) is a
  `CapacityLedger` (all-or-nothing reservations), the quote (business) is the LLM slot.
